# React Quiz App

A simple and interactive quiz application built with React and Vite.

## Features

- Multiple-choice questions with timer
- Skip functionality for questions
- Summary screen showing results
- Responsive design

## Installation

1. Clone the repository:

   ```
   git clone <repository-url>
   cd react-quiz
   ```

2. Install dependencies:

   ```
   npm install
   ```

3. Start the development server:

   ```
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

- Click on the quiz to start answering questions
- Select an answer or skip the question
- View your results on the summary screen after completing all questions

## Technologies Used

- React 19
- Vite
- JavaScript (ES6+)
- CSS

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
